# DAEGU NIGHT LIFE - 대구 룸

대구 주점 안내 웹사이트 - Next.js + Tailwind CSS

## 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 브라우저에서 열기
# http://localhost:3000
```

## 빌드

```bash
npm run build
npm start
```

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (아이콘)
